# AI LearnHub

A responsive landing page for an AI/ML online learning platform — built with semantic HTML5 and CSS3.

## Live Preview
Open `index.html` in a browser, or run with VS Code's Live Server extension.

## Features
- Sticky navigation with responsive collapse
- Hero section with animated floating image and stat highlights
- Feature cards (flexbox)
- Course catalog (CSS Grid, `auto-fit` responsive columns)
- About section with image + content split layout
- Stats band with gradient background
- Testimonials grid
- Contact section with styled form
- Footer with grid-based link columns and social icons
- Fully responsive: breakpoints at 992px, 768px, and 576px

## Tech Stack
- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, CSS variables, keyframe animations)
- Google Fonts (Poppins)
- No frameworks, no JavaScript dependencies
